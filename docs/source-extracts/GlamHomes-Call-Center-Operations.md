Agent 3: +1 (626) 702 3820 Agent 4: +1 (240) 852 1874 Agent 5: +1 (218) 559 7715 Agent 6: +1 (656) 213 1016 Agent 7: +1 (251) 501 2899 Agent Roles & Responsibilities Agent 1 — Property Concierge (pre-booking) Number: +1 (774) 485 6822 Role: This is the agent that has the full mansion catalog and helps choose the best option and move to the next action (quote/link/handoff). What it does (current operation): Greets with transparency + human option (reduces fraud suspicion). Opens with early branching: “specific villa or recommendation?” If they are not sure, it uses guided discovery with quick questions (dates, guests, area, must- haves, optional budget). Keeps a conversational talk ratio: short prompts, one question at a time, no monologues, clear next step. Recommends 1–3 options in a usable way on the call: human description, without addresses, without using the number as the “main name.” Immediate links when requested (no loops): send → confirm received → then ask for the minimum if needed to quote. Detects closing signals (“just send me links”) and closes cleanly: send → confirm → follow-up → close. Firm rules: does not invent availability/totals and does not share addresses/codes pre-booking; uses catalog only. Agent 2 — Post-Booking Check-in Concierge (post-booking, before check-in) Number: +1 (934) 220 4441 Role: It activates after booking. It already brings the full reservation record (it is transferred from the booking to BAPI). What it does: Welcomes, confirms dates/guests and schedules. Provides check-in guidance without revealing restricted data if it is not ready. Resolves basic questions about that property and closes with the support number. Does not collect phone/email (because it already comes from the booking).

Agent 3 — Check-in Welcome & Stay Kickoff (at check-in / right after) Number: +1 (626) 702 3820 Role: Confirms whether they have already checked in, gives a real welcome, and “starts” the stay. What it does: Asks: “Have you already completed your check-in and are you inside?” If yes: welcome + brief guidance (key amenities, key rules, how to ask for help). If not: minimal help + escalates if they are stuck getting in. Establishes follow-up (“I’ll call you on such day to see how your experience is going”). Agent 4 — Maintenance & Troubleshooting Concierge (maintenance) Number: +1 (240) 852 1874 Role: Support during the stay for issues like “it doesn’t work / something got stuck.” What it does: Quick diagnosis and guided basic solutions (lock, blinds, control, WiFi/TV, AC, plumbing, light, appliances). If it is not resolved in a few steps, or it is urgent: escalates to operations/provider (without promising times). Priority: safety + calm + solve quickly. Agent 5 — Stay Experience & Feedback Follow-up (quality/well-being) Number: +1 (218) 559 7715 Role: Experience follow-up to collect opinions and detect problems. What it does: Asks how the stay is going, what was the best part, and what can be improved. If it detects a problem: Maintenance → sends to Agent 4 Check-in/access → sends to Agent 3 Serious conflict → human/ops Soft reminder: at the end, the platform will ask them for an evaluation/review. Agent 6 — Checkout Reminder & Departure Help (departure reminder) Number: +1 (656) 213 1016 Role: Remind them that tomorrow is check-out and resolve departure questions.
